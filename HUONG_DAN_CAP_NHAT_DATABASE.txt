HƯỚNG DẪN CẬP NHẬT DATABASE CHO HỆ THỐNG ĐƠN HÀNG
================================================================

CÁCH 1: SỬ DỤNG PHPMyADMIN (KHUYẾN NGHỊ)
----------------------------------------

1. Mở trình duyệt và truy cập: http://localhost/phpmyadmin

2. Chọn database "dienlanh_shop" từ danh sách bên trái

3. Chọn tab "SQL" ở phía trên

4. Copy toàn bộ nội dung từ file "model/quick_update.sql" và paste vào ô SQL

5. Nhấn nút "Go" hoặc "Thực thi" để chạy

6. Kiểm tra kết quả - sẽ hiển thị "Cập nhật thành công!"

CÁCH 2: SỬ DỤNG FILE SQL ĐẦY ĐỦ
--------------------------------

1. Mở file "model/update_database.sql" trong phpMyAdmin
2. Chạy từng phần hoặc toàn bộ file
3. File này sẽ cập nhật đầy đủ hơn với dữ liệu mẫu

CÁCH 3: SỬ DỤNG COMMAND LINE (CHO NGƯỜI DÙNG NÂNG CAO)
-----------------------------------------------------

1. Mở Command Prompt
2. Di chuyển đến thư mục MySQL: cd D:\xampp\mysql\bin
3. Chạy lệnh: mysql -u root -p dienlanh_shop < D:\xampp\htdocs\project\model\quick_update.sql

KIỂM TRA SAU KHI CẬP NHẬT:

LƯU Ý QUAN TRỌNG:
-----------------

- File này sử dụng "CREATE TABLE IF NOT EXISTS" nên sẽ không xóa dữ liệu hiện có
- File này sử dụng "INSERT IGNORE" nên sẽ không tạo trùng lặp tài khoản
- Đảm bảo XAMPP đang chạy trước khi thực hiện
- Backup database trước khi cập nhật (khuyến nghị)

4. Xem thông báo lỗi trong phpMyAdmin
